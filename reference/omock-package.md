# omock: Creation of Mock Observational Medical Outcomes Partnership Common Data Model

Creates mock data for testing and package development for the
Observational Medical Outcomes Partnership common data model. The
package offers functions crafted with pipeline-friendly implementation,
enabling users to effortlessly include only the necessary tables for
their testing needs.
